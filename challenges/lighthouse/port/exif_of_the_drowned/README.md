# EXIF of the Drowned

The fishermen of Tideglass dredged a sealed brass case from the wreck of the *Maribel* last week. Inside: a single exposed photographic plate, its image somehow intact — a grainy shot of a half-submerged hull, ribs reaching up out of the surf like a drowned thing's fingers.

The plate's metadata, miraculously, survived the salt. Whoever took this picture wanted someone to find it. They left a note buried in the file itself, in a place no casual viewer would think to look.

Examine the photograph. The Treasure is written in the metadata.

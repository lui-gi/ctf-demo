# The Drowned Admin

The wreck of the *HMS Hesperus* washed up on Smuggler's Reef with a single waterlogged terminal still flickering in the captain's quarters. Salt has eaten most of the hull, but the harbourmaster's login portal still answers requests — barely. Locals say the old admin, "Drowned Eli," was rumored to keep his Treasure pinned right on the dashboard banner where any boarder could see it, if only they could get past the door.

Your Crew rigs a longboat to the Hesperus and points a browser at the rotting login form. The server coughs back ugly errors the moment you so much as breathe a quote at it. Eli, gods rest him, never learned to escape an apostrophe.

Slip past the login. Whatever Eli was bragging about lives on the admin dashboard.

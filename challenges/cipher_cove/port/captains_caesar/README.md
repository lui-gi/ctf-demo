# The Captain's Caesar

The first thing the deckhands found in Captain Lazarus Crowfoot's sea-chest was a single scrap of parchment, brittle and salt-stained, scrawled in his own crabbed hand. The letters meant nothing — gibberish, a drunkard's alphabet — but at the top of the page, in a tidier script, the captain had written a small joke: *"Caesar's old trick still serves an honest pirate."*

Crowfoot was a gambler and a fool, but he was never one for elaborate ciphers. Whatever he encoded here, he encoded shallow.

# Message in a Bottle

The tide brought it in at sunrise: a corked green bottle wedged between two rocks, and inside the bottle, a single photograph of itself bobbing on a beach. The photo had been folded twice and sealed with wax. Whoever set it adrift wanted it to be found, but they didn't want it read by just anyone who happened to glance at it.

The image itself looks ordinary. But pirates know — what's *behind* the picture matters as much as what's in it.

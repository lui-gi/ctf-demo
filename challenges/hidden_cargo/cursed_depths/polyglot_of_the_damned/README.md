# Polyglot of the Damned

A single file washed up in the bilge of the *Last Reckoning*. PNG header. Loads in any image viewer. The crew shrugged.

But when our archivist dragged it into a ZIP utility, it opened. When she pointed her PDF reader at it, that opened too. The same bytes, three identities. And inside each identity, a different fragment of something larger — a single Treasure split across the three readings of the file plus a fourth fragment that only reveals itself when you XOR the first three together against a constant the file itself carries.

You'll need every interpretation. Treat the file as PNG, as ZIP, and as PDF, in turn. Then reassemble.

## Files
- `polyglot.bin` — a single file that is simultaneously a valid PNG, ZIP, and PDF

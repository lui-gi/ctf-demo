Cargo Manifest Validator v2 is now in production

2026-04-12 · by Ines Tidemark

Today we shipped v2 of marrowtide-internal/manifests-cli, our internal tool for validating cargo manifests against our underwriting policy.

The package is published to our internal registry at npm.marrowtide.example. If you have a Marrowtide engineering token (we do not currently issue tokens to anyone outside the company) you can npm install it directly.

The headline change is that v2 validates the entire manifest in a single pass; v1 needed three. Most of our manifests are now under 80 ms end-to-end.

Where the source lives

The repository sits at git.marrowtide.example/manifests-cli. The repo itself is private — the public landing page tells you what is in it but the source requires auth.

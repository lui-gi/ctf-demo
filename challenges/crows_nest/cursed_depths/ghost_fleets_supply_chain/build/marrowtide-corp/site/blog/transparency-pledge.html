Our transparency pledge

2026-02-08 · by Sela Marrowfen

Marrowtide commits to four public surfaces:

  1. An internal package registry whose package list is publicly searchable, even though most packages require an engineering token to install.
  2. A self-hosted git host whose repository directory is public, even though most repositories are private.
  3. A certificate transparency log mirror listing every certificate ever issued under *.marrowtide.example.
  4. A .well-known/security.txt on every hostname we operate, listing how to disclose vulnerabilities.

If we ever stand up a host that is not in the certificate log, that is a bug; please email security@marrowtide.example.

manifests-cli

Internal tooling for cargo manifest validation at Marrowtide Logistics. Used by the underwriting pipeline to validate every bill of lading before it is bound.

Operational notes

  • v2 ships with the underwriting pipeline as of 2026-04-12. See the engineering blog post.
  • Tests live in tests/ and run against fixtures from staging-internal.marrowtide.example (internal-only host; not reachable from the public network).
  • Every certificate we ever issue under *.marrowtide.example is published to our certificate-transparency mirror at crt.marrowtide.example. If you are auditing what hosts we operate, that is the canonical source.
